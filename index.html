Book Price API

OpenBookPrices is an API that lets you find the current prices for books at different ventures, including shipping to different countries. It will also convert the prices into the currency you request.

It is free to use and requires no authentication or sign up.

An Example

This is an example request for prices of the excellent "McGee on Food and Cooking". The response is JSON so it would not have the inline comments shown below:

{% highlight bash %} curl http://api.openbookprices.com/prices/9780340831496/US/USD {% endhighlight %} {% highlight js %} // return is an array of entries... "country": "US", "currency": "USD", "preConversionCurrency": "GBP", // Details about the vendor "vendor": { "code": "amazon_uk", "name": "Amazon UK", "homepage": "http://www.amazon.co.uk/" }, // The url that the book can be purchased from. "url": "http://www.amazon.co.uk/dp/0340831499", // The url to the API endpoint for this vendor, to be used if price data // is not cached.
