Book Price API

OpenBookPrices is an API that lets you find the current prices for books at different vendors, including shipping to different countries. It will also convert the prices into the currency you request.

It is free to use and requires no authentication or sign up.

An Example

This is an example request for prices of the excellent "McGee on Food and Cooking". The response is JSON so it would not have the inline comments shown below:

"offers": { "new": { "price": 35.08, "availabilityNote": "Usually dispatched within 24 hours", "shippingNote": "5 to 7 business days", "shipping": 11.28, "total": 46.36 } }, // Details of the vendor useful for when displaying these prices. "meta": { // "timestamp" and "ttl" can be used to determine freshness (same values // are used to set the caching headers in the HTTP response). "timestamp": 1382165629, "ttl": 3600, // "retryDelay" is a value in seconds to wait before requesting the // prices again which is easier to work with than adding logic based // on "status". If not retry is needed it's value will be falsy. "retryDelay": null, // "preConversionCurrency" is the currency that the price was originally // in on the vendor's site before it was converted into the requested // currency (in this case a conversion has been made from British Pounds // to US Dollars).
